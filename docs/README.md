# devine Documentation

This folder contains project-level documentation used by engineers, designers, and coding AI agents.

## Files

- `project-information.md`
  - Detailed project context, goals, core loops, and product constraints.
- `tech-stack.md`
  - Current and planned technical stack for app, backend, AI, and analytics.
- `theme-and-design-system.md`
  - Visual direction, feminine-leaning premium theme, color system, and UI rules.
- `agent-rules/coding-agent-rules.md`
  - Required coding behavior for autonomous and assisted AI agents.
- `agent-rules/implementation-checklist.md`
  - Build and review checklist agents must pass before shipping changes.

## Priority Order for Rules

If two rules conflict, follow this order:
1. Repository root `AGENTS.md`
2. `docs/agent-rules/coding-agent-rules.md`
3. Feature-specific docs and specs
